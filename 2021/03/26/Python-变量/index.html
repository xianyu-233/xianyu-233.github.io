Python的变量

在Python中，变量没有特定的标识符，即定义变量时不需要用特定的标识符来修饰变量。

变量的使用规范：

  • 变量名只能包含字母、数字和下划线。变量名可以用下划线和字母开头，但不能用数字开头。
  • 变量名不能包括空格。
  • 变量名不能与Python中的函数名与关键字重名。
  • 变量名尽量为小写。

一些奇奇怪怪的错误
  • 上面也说了，Python在定义变量时不需要标识符，字符串又可以用+来拼接，那么在拼接字符串时塞个数字进去会怎样？如message = "A"+23+"rd Years"
  • 答案是会报错，因为Python无法识别出这个拼接后的message是什么类型的数据。
  • 这是可以用str()来避免这个错误，即message = "A"+str(23)+"rd Years"
