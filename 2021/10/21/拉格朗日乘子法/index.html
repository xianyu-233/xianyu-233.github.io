拉格朗日对偶性介绍

在求解有约束的最优化问题时，经常使用的方法是：利用拉格朗日对偶性将原始问题转化为对偶问题，通过求解对偶问题从而得到原始问题的解。

原始问题

定义：

假设$f(x),c_i(x),h_j(x)$是定义在$R^n$上的连续可微函数，约束最优化问题为：
$$min_{x \in R^n} \ \ f(x)$$

$$s.t. \quad c_i(x) \le 0, \quad i=1,2,…,k$$

$$\quad h_j(x)=0,\quad j=1,2,…,l$$

  • 首先，我们引入拉格朗日函数：

    $$L(x,\alpha,\beta)=f(x)+\sum^K_{i=1}{\alpha_i}{c_i(x)}+{\sum}^M_{j=1}\beta_jh_j(x)$$

    其中，$x=(x^{(1)},x^{(2)},…,x^{(n)})^T \in R^n$，$\alpha_i,\beta_j$是拉格朗日乘子，$\alpha_i \ge 0$。考虑x的函数：
    $$\theta_P(x)=max_{\alpha,\beta;\alpha_i \ge 0} \ \ L(x,\alpha,\beta)$$
    这里，$\theta_P(x)$表示原始问题$f(x)$+约束条件。

  • 引入拉格朗日函数的作用：

    我们知道，要求解带约束的优化问题，梯度下降，牛顿法这类方法都是不能解决的，因此我们可以通过拉格朗日函数，将约束条件融入到问题上，即将约束条件塞到$f(x)$里，构成一条式子。
    当x不满足约束条件时，即某个i使得$c_i(w) > 0$或者是某个j使得$h_j(w) \neq 0$,那么就有
    $$\theta_P(x)=max_{\alpha,\beta;\alpha_i \ge 0} \ \ [f(x)+\sum^K_{i=1}{\alpha_i}{c_i(x)}+{\sum}^M_{j=1}\beta_jh_j(x)]=+\infty$$
    原因：

    • 当$c_i(w) > 0$时，函数的目的是取到最大值，那么我们可以直接令这个i对应的$\alpha_i = +\infty$，这里取最大值的结果会变成$+\infty$。
    • 同理，如果$h_j(w) \neq 0$，函数的目的是取到最大值，那么j对应的$\beta_j= +\infty$，这里取最大值的结果会变成$+\infty$。

    因为这里的目的是取最大值，而且$c_i(w) \le 0,\alpha_i \ge 0$（负数乘一个正数，结果还是负数）且$h_j(w) = 0$，那么$\theta_P(x)$的最大值就等于$f(x)$，所以可得，
    $$\theta_P(x)=\begin{cases}f(x) \quad \quad x满足约束 \\+\infty \quad \quad 其他 \\\end{cases}$$

    因此原问题的定义域就扩展到了整个实数域，不过只有满足约束条件的才等于$f(x)$，不满足的那一部分都等于正无穷。

    如果考虑求原问题的最小化，那么可以得到：
    $$min_x \ \theta_P(x)=min_x \ max_{\alpha,\beta;\alpha_i \ge 0} \ \ L(x,\alpha,\beta)$$
    到此，我们已经将原问题的形式，从
    $$min_{x \in R^n} \ \ f(x)$$

    $$s.t. \quad c_i(x) \le 0, \quad i=1,2,…,k$$

    $$\quad h_j(x)=0,\quad j=1,2,…,l$$
    变成了
    $$min_x \ \theta_P(x)=min_x \ max_{\alpha,\beta;\alpha_i \ge 0} \ \ L(x,\alpha,\beta)$$

    • 这样一来，求出$min_x \ \theta_P(x)$的解就等于求出了$min_{x \in R^n} \ \ f(x)$的解。
      设原始问题的最优值为：$p^*=min_x \ \theta_P(x)$

    补充：

    • 后面一直在提的原始问题也是$min_x \ \theta_P(x)$这个形式。
    • 这个问题又称为广义拉格朗日函数的极小极大问题。

对偶问题

定义：

$$\theta_D(\alpha,\beta)=min_x \ \ L(x,\alpha,\beta)$$

  • 再考虑极大化$\theta_D(\alpha,\beta)=min_x \ \ L(x,\alpha,\beta)$，即

$$max_{\alpha,\beta:\alpha_i \ge 0} \ \ min_x \ L(x,\alpha,\beta)$$

  • 该问题又称为广义拉格朗日函数的极大极小问题。

此时我们可以将广义拉格朗日函数的极大极小问题表示为约束最优化问题：

$$max_{\alpha,\beta} \ \ min_x \ L(x,\alpha,\beta)$$
$$s.t. \quad \alpha_i \ge 0, \quad i=1,2,…,k$$

同时我们也可以将其称为原始问题的对偶问题，定义对偶问题的最优值：

$$d^*=max_{\alpha,\beta:\alpha_i \ge 0} \ \ \theta_D(\alpha,\beta)$$

原始问题与对偶问题的关系

定理：
如果原问题和对偶问题都有最优解$p^*$和$d^*$。
那么它们满足：

$$d^* = max_{\alpha,\beta:\alpha_i \ge 0} \ \ \theta_D(\alpha,\beta) \le min_x \ \theta_P(x)=p^*$$
证明：

  • 因为$\theta_D(\alpha,\beta)=min_x \ L(x,\alpha,\beta)$而$\theta_P(x)=max_{\alpha,\beta:\alpha_i \ge 0} \ L(x,\alpha,\beta)$
  • 所以对于任意变量恒有，$\theta_D(\alpha,\beta) \le theta_P(x)$
  • 那么容易得到，$max_{\alpha,\beta:\alpha_i \ge 0} \ \theta_D(x,\alpha,\beta) \le min_x \ theta_P(x,\alpha,\beta)$
  • (对于任何x，恒有$f(x) \ge g(x)$，那么在$g(x)$中取得的最大值也会小于等于$f(x)$中取得的最小值)，这就是极大值中的极小值恒大于极小值中的极大值。
  • 所以有$d^* \le p^*$

设$x^*,\alpha^*,\beta^*$分别是对偶问题$\theta_D$和原始问题$\theta_P$的可行解，且$d^* = p^*$，那么$x^*,\alpha^*,\beta^*$就是原始问题与对偶问题的最优解。
在某些条件下，原始问题与对偶问题的解是相等的，即$d^* = p^*$。那么就可以利用对偶问题的解来代替原始问题的解了。

  • 定理一：、

    若$f(x)$和$c_i(x)$是凸函数，$h_j(x)$是仿射函数，$c_i(x)$是严格可行的，即存在x对于所有i都有$c_i(x)<0$,那么存在$x^*,\alpha^*,\beta^*$，$x^*$是原始问题的解，$\alpha^*,\beta^*$是对偶问题的解，并且
    $$p^*=d^*=L(x^*,\alpha^*,\beta^*)$$

  • 定理二（KKT条件）：
    在满足定理一的基础上，$x^*,\alpha^*,\beta^*$，$x^*$分别是原始问题和对偶问题的解的充分必要条件是，$x^*,\alpha^*,\beta^*$，$x^*$满足以下条件：

    • $\nabla_x L(x^*,\alpha^*,\beta^*)=0$
    • $a_i^* c_i(x^*)=0, \quad \quad i=1,2,…,k$
    • $c_i(x^*) \le 0, \quad \quad i=1,2,…,k$
    • $a_i^* \ge 0, \quad \quad i=1,2,…,k$
    • $h_j(x^*)=0 \quad \quad j=1,2,…,l$

    其中根据$a_i^* c_i(x^*)=0, \quad \quad i=1,2,…,k$（称为KKT互补条件）可以得到：若$a_i^* > 0$，则$c_i(x^*) = 0$。
