Numpy介绍

  • Numpy(Numerical Python)是Python语言的一个扩展程序库。
  • Numpy支持大量的维度数组与矩阵运算，同时也有大量的针对数组运算的数学函数库。

Ndarray对象

在Numpy中最重要的一个对象是，N维数组对象ndarray。
ndarray内部由以下内容组成：

  • 一个指向数据的指针。
  • 数据类型或dtype，描述在数组中的固定大小值的格子。
  • 一个表示数组形状的元组，表示各维度大小的元组。
  • 一个跨度元组，其中的整数指的是为了前进到当前维度下一个元素需要“跨过”的字节数。

创建一个ndarray只需调用Numpy的array函数即可。

参数说明：

  • object $\quad \quad$数组或嵌套的数列，必选
  • dtype $ \ \quad \quad $数组元素的数据类型，可选
  • copy $ \ \ \quad \quad $对象是否需要复制，（默认为True），可选
  • order $\ \quad \quad $创建数组的样式，C为行方向，F为列方向，A为任意方向（默认），可选
  • subok $\ \quad \quad $默认返回一个与基类类型一致的数组（默认False），可选
  • ndmin $\ \quad \quad $指定生成数组的最小维度，可选

Numpy数据类型

numpy支持的数据类型比Python自带的类型要多，基本上可以和C语言的数据类型对应上，下列为Numpy的基本数据类型

  • bool$\quad$布尔型数据类型
  • int $\quad$默认的整数类型
  • intc$\quad$与C的int型
  • intp$\quad$用于索引的整数类型
  • int8$\quad$字节（-128 到 127）
  • int16$\quad$字节（-32768 到 32767）
  • int32$\quad$字节（-2147483648 到 2147483647）
  • int64$\quad$字节（-9223372036854775808 到 9223372036854775807）
  • uint8$\quad$无符号整数（0 到 255）
  • uint16$\quad$无符号整数（0 到 65535）
  • uint32$\quad$无符号整数（0 到 4294967295）
  • uint64$\quad$无符号整数（0 到 18446744073709551615）
  • float$\quad$ float64类型的简写
  • float16$\quad$半精度浮点数，包括：1 个符号位，5 个指数位，10 个尾数位
  • float32$\quad$半精度浮点数，包括：1 个符号位，8 个指数位，23 个尾数位
  • float64$\quad$半精度浮点数，包括：1 个符号位，11 个指数位，52 个尾数位
  • complex$\quad$ complex128 类型的简写，即 128 位复数
  • complex64$\quad$ 复数，表示双 32 位浮点数（实数部分和虚数部分）
  • complex128$\quad$ 复数，表示双 64 位浮点数（实数部分和虚数部分）

注：其实就是bool、int、uint、float和complex这几类数据。

Numpy常用的创建函数

属性 描述
np.ones((m:n),dtype) 生成m行n列全1的矩阵数组（需要指明数据类型dtype）
np.ones_like(a) 生成数组a形状的全1数组
np.zeros((m:n),dtype) 生成m行n列全0的矩阵数组(需要指明数据类型dtype)
np.ones_zeros(a) 生成数组a形状的全0数组
np.full((m,n),val) 生成m行n列，值为val的矩阵数组
np.full_like(a,val) 生成数组a形状，值为val的矩阵数组
np.eye(n) 生成n阶单位矩阵
np.diag([a]) 生成以a值为主对角线，其他值为0的矩阵数组（存在问题，不建议使用）
np.linspace(start,end,num) 生成等间距排列的num个元素对象的数组（从start开始到end结束）
