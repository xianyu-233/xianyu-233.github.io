数据类型概述

在Python$ \ $3.x的版本中，主要有7种数据类型：

  • 数值型（整型int、浮点型float、复数complex、布尔型bool）
  • 字符串（string）
  • 字节（bytes）
  • 列表（list）
  • 元组（tuple）
  • 集合（set）
  • 字典（dict）

根据数据是否拥有哈希值，可以将上述的7种数据类型分成可变数据类型和不可变数据类型。

  • 可变数据类型：列表、字典、集合
  • 不可变数据类型：数值型、字符串、字节、元组
    补充：哈希值是一种有损压缩数据的格式，它可以把任意长度的数据通过通过散列压缩成固定长度的输出，这个输出值就是哈希值。

数值

在Python$ \ $3.x中数值型数据包括整型int、浮点型float、复数complex、布尔型bool

整型

在Python中，可以定义处理任意大小的整数（无论多少位数，没错就算100位也行），包括负整数。其语法与书写的整数一致，如a=100、b=-10000等。同时，Python中也支持用十六进制来表示整数（注意要加0x的前缀），如a = 0xff0等。

浮点型

在Python中，浮点型只有float一类，而且小数点之后最多可以显示15位。
浮点型的数既可以用通常的显示方法，如a = 1.2345，也可以用科学记数法，如a = 1.6e3=1600.0，其中e表示10.

复数
  • 复数是由实部与虚部组成，可以用$a+bj$来表示，也可以用complex(a,b)表示。其中a和b都是实数（浮点型）。
  • 注意：在Python中虚数不可能单独存在，并且在虚部中必须加上后缀j或J。
  • 对于复数来说，可以通过real()和imag()来分别调用复数的实部和虚部。
  • 对于复数，Python中可以用conjugate()返回复数的共轭复数。
