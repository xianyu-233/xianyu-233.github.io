牛顿法介绍

牛顿法是求解无约束最优化问题的常用方法之一，有收敛速度快的优点。
牛顿法属于迭代算法，每一步都需要求解目标函数的二阶导数（hessian矩阵）
具体实现过程：

非向量形式

目标：找到一个有二阶导数函数$f(x)$的零点。

  • 首先在零点附近取一点$x_0$进行泰勒展开，即
    $$f(x)=f(x_0)+f’(x_0)(x-x_0)+ \frac{1}{2}f’’(x_0)(x-x_0)^2$$
  • 由于求的是零点，所以$f’(x)=0$，即对上式求导
    $$f’(x) = f’(x_0)+f’’(x_0)(x-x_0)=0$$
  • 化简可得，
    $$x = x_0 - \frac{f’(x_0)}{f’’(x_0)}$$
  • 每次更新一次x都可以迭代一次，所以最终可得
    $$x_{n+1} = x_n - \frac{f’(x_n)}{f’’(x_n)}$$

因此只要一直迭代下去就可以找到$f(x)$的零点，而且下降速度是建立在二阶导上的，所以下降速度会非常快。

向量形式

目标：找到一个有二阶导数函数$f(X)$的零点。
不一样的是，这里的$X=[x_1,x_2,…,x_n]$的一个$n \times 1$的向量矩阵

  • 首先也是在零点附近取一点$X_0$进行泰勒展开，即
    $$f(X) = f(X_0)+g(X_0)(X-X_0)+\frac{1}{2}(X-X_0)^T H (X-X_0)$$

    • 其中，$g(X)=[\frac{\partial f(X)}{\partial x_1},\frac{\partial f(X)}{\partial x_2},…,\frac{\partial f(X)}{\partial x_n}]$
    • H是一个hessian矩阵,具体值如下

    $$\begin{bmatrix}
    {\frac{\partial^2 f(X)}{\partial x_1 \partial x_1}}&{\frac{\partial^2 f(X)}{\partial x_1 \partial x_2}}&{\cdots}&{\frac{\partial^2 f(X)}{\partial x_1 \partial x_n}}\\
    {\frac{\partial^2 f(X)}{\partial x_2 \partial x_1}}&{\frac{\partial^2 f(X)}{\partial x_2 \partial x_2}}&{\cdots}&{\frac{\partial^2 f(X)}{\partial x_2 \partial x_n}}\\
    {\vdots}&{\vdots}&{\ddots}&{\vdots}\\
    {\frac{\partial^2 f(X)}{\partial x_n \partial x_1}}&{\frac{\partial^2 f(X)}{\partial x_n \partial x_2}}&{\cdots}&{\frac{\partial^2 f(X)}{\partial x_n \partial x_n}}\\
    \end{bmatrix}$$

  • 同样$f’(X) = 0$,得到$g(X_0)+H(X-X_0)=0$

  • 化简得，$X = X_0-H^{-1}g(X_0)$
    补充：当H为正定矩阵时，$f(X)$可以找到极小值，当H为负定矩阵时，$f(X)$可以找到极大值，当H为不定矩阵时，$f(X)$找不到极值，当H为半正（负）定矩阵时，$f(X)$不确定是否有极值，需要通过其他方法来找极值。

在向量形式的牛顿法中，计算量最大在于求出hessian矩阵的逆($H^{-1}$)，对此拟牛顿法就是找一个同为n阶的方阵$G_k$来近似代替$H^{-1}$，从而减少计算量。

牛顿法的下降保证

如果$H_k$是正定的（$H_k^{-1}$也是），那么可以保证牛顿法搜索方向$p_k$是下降方向。
原因：

由$X^{k+1} = X^k - H_k^{-1}g_k$可得下降的方向$p_k=-g_k$
所以$f(X)$在$X^{k}$处进行泰勒展开（展开一项）得，
$$f(X) = f(X^k)+g_k(X-X^k)$$
化简成，
$$f(X) = f(X^k)-g^T_k H^{-1}_k g_k)$$
因为$H^{-1}_k$是正定的，所以$g^T_k H^{-1}_k g_k$恒大于0，所以$f(X) < f(X^k)$
所以牛顿法是下降的。

拟牛顿法

在牛顿法中，每一次迭代都需要重新计算一个$n \times n$的hessian矩阵的逆，因此在特征值较大时会造成很大的计算量，而且hessian阵还必须是正定矩阵。因此在牛顿法的技术上加以改进，利用一个同为$n \times n$的正定矩阵$G_k$或$B_k$来替代hessian矩阵。
首先，先来看拟牛顿法的hessian矩阵在迭代过程所满足的条件：

$$\nabla f(X) = g_k + H_k(X-X^{k})$$
令$X = X^{k+1}$
$$g_{k+1} - g_k + H_k(X^{k+1}-X^k)$$
令$g_{k+1} - g_k=y_k$,$X^{k+1}-X^{k} = \delta$，则$H_k$满足以下条件
$$y_k=H_k \delta_k$$
或
$$H_k^{-1}y_k = \delta_k$$

以上的称为拟牛顿条件。
拟牛顿法是将$G_k$近似代替$H_k^{-1}$或将$B_k$近似代替$H_k$
按照牛顿法条件，我们在每次迭代中选择更新$G_{k+1}$，
$$G_{k+1}=G_k+\Delta G_k$$

DFP算法

根据拟牛顿法要满足的条件，$G_{k+1}$应该满足：$G_{k+1}y_k = \delta_k$
DFP算法中，$G_{k+1}$的迭代是由$G_k$加上两个附加矩阵$P_k$和$Q_k$，即
$$G_{k+1}=G_k+P_k+Q_k$$
两边乘上$y_k$
$$G_{k+1}y_k=G_k y_k+P_k y_k+Q_k y_k$$
为了让$G_{k+1}$满足拟牛顿的条件，所以可以使$P_k$和$Q_k$满足，
$P_k y_k = \delta_k$和$Q_k y_k = -G_k y_k$
所以$P_k$和$Q_k$可以写成，
$$P_k = \frac{\delta_k \delta_k^T}{\delta^T_k y_k}$$
$$Q_k = -\frac{G_k y_k y_k^T G_k}{y_k^T G_k y_k}$$
所以$G_{k+1}$的迭代公式为：

$$G_{k+1} = G_k + \frac{\delta_k \delta_k^T}{\delta^T_k y_k} - \frac{G_k y_k y_k^T G_k}{y_k^T G_k y_k}$$

可以被证明的：如果初始的$G_0$（自己设的矩阵）是正定矩阵，那么经过迭代后的$G_n$也是正定矩阵。
