    补充：这个函数可以对任何字符串数据使用，并且rstrip或lstrip指的是删除右边的或左边的空格，strip则是两边空格都删掉。

    • 绝对路径：从基本盘的位置开始到文件所在的位置，一般开头为/或’X:'（X代表盘号）

结果：

  • open函数中的第二个参数可以指定打开文件的方式，其中默认为r读取文件，w为写入文件（覆盖地写入），a为附加模式（可以在原来的基础上继续追加数据），r+是同时允许读取和写入文件。
    多行写入：

    多行写入只需多次调用file_data.write(a)即可。
    注意：多行写入也是覆盖式写入，而且需要换行的话要自行在每行末尾追加\n

附加写入

将open()中第二个参数改为a后，python不会在返回文件对象前清空文件。
你写入到文件的行都将追加到文件末尾。
