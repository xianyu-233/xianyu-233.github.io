什么是Markdown：

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。
Markdown 编写的文档后缀为 .md或 .markdown。

Markdown的应用

Markdown 能被使用来撰写电子书，如：Gitbook。
当前许多网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上发表消息。
例如：GitHub、简书、reddit、Diaspora、Stack Exchange、OpenStreetMap 、SourceForge等。

一、Markdown的标题

1、使用#来表示不同的标题

语法如下：（注意#后面有空格）

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
(好像有六级标题，但第六级标题与第五级标题差不多，所以就懒得写了)

效果如下：

一级标题

二级标题

三级标题

四级标题

五级标题

2、使用=和-来表示不同的标题

语法如下：

这是一级标题
=========
这是二级标题
-——–

效果如下：

这是一级标题

这是二级标题

二、Markdown的段落格式

1、字体

用-或*包裹要标注的文字
一个表示斜体
两个表示粗体
三个表示斜粗体

具体语法如下：

*文字* 或 _文字_
**文字** 或 __文字__
***文字*** 或 ___文字___

效果如下：

斜体文本
斜体文本
粗体文本 （说好的粗体字呢？？）
粗体文本
粗斜体文本
粗斜体文本

具体语法：

***
* * *
******
- - -
-——–

3、删除线

在文字两端加上两个波浪线~~即可

具体语法：

啥东西都没有
~~啥东西都没有~~

效果如下：

Markdown 脚注的格式如下：

三、Markdown列表

1、无序表

无序表使用星号*、加号+或者是减号-作为标记（符号后面一定要有空格！）

具体语法

* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

效果如下：
  • 第一项
  • 第二项
  • 第三项
  • 第一项
  • 第二项
  • 第三项
  • 第一项
  • 第二项
  • 第三项

2、有序列表

有序列表使用数字加.来表示，如

1. 第一项
2. 第二项
3. 第三项

效果如下：
  1. 第一项
  2. 第二项
  3. 第三项

四、Markdown区块

Markdown区块是在开头使用>符号，然后紧跟一个空格，如：

> 区块引用
> Markdown学习
> 一直摸鱼

结果如下：

区块引用
Markdown学习
一直摸鱼

当然，区块也是可以嵌套的，
一个>是最外层，两个为第一层嵌套，以此类推：

>最外层
>>第一层嵌套
>>>第二层嵌套

结果如下：

最外层

第一层嵌套

第二层嵌套

六、Markdown链接

链接使用方法如下：

[链接名称](链接地址)
或者
<链接地址>

用[]()来表示：

例如：这是一个链接[百度](https://www.baidu.com)
效果：这是一个链接百度

直接使用链接地址：

例如：<https://www.baidu.com>
效果：https://www.baidu.com

当然也可以博客内文章之间的链接：

例如：转去另一篇[博客](\2021\01\01\Markdown的语法-续/)
效果：转去另一篇博客

七、Markdown图片

Markdown图片语法格式如下：

![alt 属性文本](图片地址)
![alt 属性文本](图片地址 “可选标题”)
（这个图片地址是要从服务器最开始的地址开始写起）

  • 开头一个感叹号 !
  • 接着一个方括号，里面放上图片的替代文字
  • 接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字。
实例：

kano 的图片

八、Markdown表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。
语法格式如下：

| 表头 | 表头 |
| --- | --- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |

效果如下：
表头 表头
单元格 单元格
单元格 单元格

当然也可以设置表格的对齐方式：

  • -:设置内容和标题栏居右对齐。
  • :-设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
语法如下：

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

效果如下：
左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格

注意:文字和表格最好隔开一行不然表格可能会失效。

九、拓展的东西

支持的 HTML 元素

目前支持的HTML元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等

转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符：
Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\ 反斜杠
` 反引号
* 星号
_ 下划线
{} 花括号
[] 方括号
() 小括号
# 井字号
+ 加号
- 减号
. 英文句点
! 感叹号

改文字颜色与字体

要改变字体和颜色可以使用标签
要改变字体大小可以用size属性
改变颜色的化要用color属性(支持十六进制)
改颜色的例子：

<font color=blue>我是蓝色</font>
<font color=#3366FF>我是蓝色</font>

样例如下：

我是蓝色
我是蓝色

改字体大小的例子：

<font size=5>我是5号字体/
<font size=6>我是6号字体/
<font size=7>我是7号字体/

样例如下：

我是5号字体
我是6号字体
我是7号字体
