一、创建github仓库

点进去以后有一个必填项就是仓库名了，其他都是可选项，填写完后可以直接找到最底下点击创建。

创建完后记住这个地址，这个就是仓库的地址了（后面会用到）。

二、使用git上传文件

顺序：

绑定账号（新安装Git时用）：

  • 为Github账号设置ssh key（新安装Git时用）：

    • 生成ssh key：

      打开 C:\Users\xxx\ .ssh，如果有这两个文件

      那么密钥已经生成，id_rsa.pub就是公钥（复制好里面的内容）

      如果没有可以通过 ssh-keygen -t rsa -C “xxx”（xxx代表邮箱）来生成

    • 为Github账户配置ssh key

      • 打开Github里的Settings：

      • 将id_rsa.pub里的内容粘贴进去

      • 这样就算完成第一步了

  • 创建本地仓库

    • 随便创建一个文件夹，并用git控制台打开这个文件

    • 利用“git init”创建一个本地仓库

    • 将文件添加进仓库（实际上可以直接用”git add .“来将当前文件夹中所有文件添加到仓库）

    • 把文件提交到仓库中：git commit -m “提交文件”（双引号内的是提交注释，一定要写的）

    • 这样本地仓库算是创建好了

  • 关联Github仓库

    • 找到并复制仓库地址

    • 执行关联命令：git remote add origin xxx(刚刚复制的地址)

  • 上传本地代码

    执行命令：git push -u origin master

三、后记

如果要将新的文件上传到GitHub上：重复git add xxx（文件名）、git commit -m “提交文件”、git push -u origin master（上传文件）即可。

注意：如果要上传到不同的仓库的时候，则需要重新关联一个仓库。
